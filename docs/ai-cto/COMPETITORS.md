# HanaNote — 竞品分析
> 最后更新: 第零轮

## 竞品概览
| 竞品 | 平台 | 核心能力 | 我们的优势 |
|------|------|----------|-----------|
| TRACE | iOS/Android | 里程碑、拍照、提醒、社区 | 无验血/PK/加密/中日文/围度分析 |
| Hormone Helper | iOS | HRT追踪、心情、拍照、医生模板 | 无加密/Android/中日文/血检/PK |
| Trans Memo | Android | 用药、库存、提醒 | 功能单一，无验血/拍照/PK |
| estrannaise.js | Web | 3室PK模型、MCMC、多酯型 | 纯Web，无App，无其他功能 |
| TFS Simulator | Web | 注射E2模拟 | 只支持注射 |
| ShotTraX | iOS | 注射追踪+社交 | 只做注射 |

## 我们的差异化
1. 全功能整合（用药+验血+围度+拍照+PK模拟）
2. 极致隐私（加密全栈+伪装+紧急清除）
3. 中日英三语（面向中日东南亚华人圈）
4. 二次元审美系统
5. 科学数据分析（三维关联分析）

## 值得借鉴
- estrannaise.js 的 3室PK模型和MCMC方法
- TRACE 的里程碑庆祝设计
- Hormone Helper 的医生沟通模板概念
- Trans Memo 的库存追踪
