# HanaNote 产品愿景
> 最后更新: 2026-03-28 | 会话轮次: #33

## 最终产品形态
面向中国大陆、日本及东南亚华人社区 16-35 岁 MTF 跨性别用户的一站式 HRT 健康记录工具。强调隐私第一、数据驱动科学用药、温暖的二次元风格体验。不是社交平台，不是远程医疗，而是记录身体变化的私密空间。

## 用户场景
核心场景：用户每天打开 app → 查看今日用药提醒 → 标记已服药 → 偶尔记录血检结果 / 身体变化 / 拍照对比 → 查看 PK 曲线了解药物浓度 → 在时间线中回顾 HRT 历程。

## 产品文档中的潜在问题
1. 缺乏首次使用引导，新用户面对 5 个空 Tab 会困惑
2. 空状态 UI 过于简陋，缺乏 CTA 引导
3. 无数据导出/备份功能的实际实现（UI 有占位按钮）
