# HanaNote 产品愿景
> 最后更新: 2026-04-26 | 会话轮次: #51

## 最终产品形态
面向中国大陆、日本及东南亚华人社区 16-35 岁 MTF 跨性别用户的一站式 HRT 健康记录工具。强调隐私第一、数据驱动科学用药、温暖的二次元风格体验。不是社交平台，不是远程医疗，而是记录身体变化的私密空间。

## 用户场景
核心场景：用户每天打开 app → 查看今日用药提醒 → 标记已服药（触发花瓣庆祝）→ 偶尔记录血检结果 / 身体变化 / 拍照对比 → 查看 PK 曲线了解药物浓度 → 在时间线中回顾 HRT 历程。

新场景（v1.2.0+）：每次启动自动检查更新（CDN 优先，GitHub fallback）→ 有新版本时弹 4 状态对话框 → 应用内下载 APK → FileProvider 触发系统安装。

## 产品文档中的潜在问题
1. **v1.1.0 Release Notes 与实际不符**：用户阅读 GitHub Release 页会看到"onboarding wizard, import, PDF"，但应用内并无对应功能。建议 R52 要么补齐功能、要么修订 Release Notes。
2. 缺少首次使用引导，新用户面对 5 个空 Tab 仍会困惑（与 R45 同样问题，未解决）。
3. 数据导入功能缺失，备份只能单向（导出可，导入不可），用户换设备无法恢复。
