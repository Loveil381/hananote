# Changelog

All notable changes to HanaNote will be documented in this file.

## [1.0.0] - 2026-04-11

### Added
- **Medication management** -- add drugs, set schedules, track daily doses with countdown timers
- **Blood test tracking** -- log hormone panels with WPATH target range indicators and trend charts
- **Body measurements** -- track 9 measurement types with historical analysis
- **Encrypted photo journal** -- end-to-end encrypted photo storage
- **Mood diary** -- daily journal entries with mood tracking and preset tags
- **Timeline** -- cross-feature chronological view with HRT milestone celebrations
- **PK Simulator** -- dual-engine (V2 + Hana-PK) pharmacokinetic modeling for 7 drug preparations
- **MAP calibration** -- calibrate PK parameters against real blood test results
- **Notification reminders** -- scheduled local notifications with global toggle
- **HRT drug reference** -- in-app WebView for hrtyaku.com evidence-based guide
- **In-app update checker** -- automatic GitHub Releases version checking
- **Privacy & Security** -- PIN + biometric auth, Argon2id + SQLCipher + AES-256-GCM encryption
- **Data export** -- JSON backup export via system share
- **Internationalization** -- Chinese, English, Japanese (390+ localization keys)
- **CI/CD** -- GitHub Actions for analysis, testing, and release builds
